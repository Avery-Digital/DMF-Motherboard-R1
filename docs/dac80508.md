# DAC80508ZRTER — 8-Channel 16-bit DAC

## Overview

The DAC80508ZRTER is an 8-channel, 16-bit voltage-output DAC with an SPI interface. It shares SPI2 with the LTC2338-18 ADC and three DRV8702 TEC drivers.

## Hardware Connection

| Signal | Pin | Port | Direction | Config |
|--------|-----|------|-----------|--------|
| nCS | 145 | PD2 | Output | Push-pull, High speed |
| MISO | 36 | PC2 | Input | (shared SPI2) |
| MOSI | 37 | PC3 | Output | (shared SPI2) |
| SCK | 128 | PA9 | Output | (shared SPI2) |

PD2 is configured by `DAC80508_Init()` via `Pin_Init()` and driven HIGH (deasserted).

## SPI Configuration

| Parameter | Value | Notes |
|-----------|-------|-------|
| Frame size | 24-bit | Temporarily reconfigured from SPI2's default 32-bit |
| SPI Mode | 1 (CPOL=0, CPHA=1) | Temporarily reconfigured from SPI2's default Mode 0 |
| Clock | 16 MHz SCK | Inherited from SPI2 prescaler (PLL3P 128 MHz / DIV8) |
| Bit order | MSB first | |
| Timeout | 5 ms | `DAC80508_SPI_TIMEOUT_MS` |

**SPI Mode notice:** The DAC80508 supports Mode 1 (CPOL=0, CPHA=1) and Mode 2 (CPOL=1, CPHA=0). This driver uses Mode 1. Verify against the DAC80508 datasheet (SLASER2) timing diagrams.

## SPI Frame Format (24-bit)

```
Bit [23]:     R/W  — 0 = write, 1 = read
Bits [22:20]: Reserved (000)
Bits [19:16]: ADDR — 4-bit register address
Bits [15:0]:  DATA — 16-bit register data
```

**Read operation:** The DAC80508 returns read data on the SPI frame FOLLOWING the read command. `DAC80508_ReadReg()` sends two transactions: (1) read command, (2) NOOP to clock out the response.

## SPI Bus Sharing

SPI2 is shared between three device types, each requiring different SPI configuration:

| Device | Data Width | SPI Mode | Notes |
|--------|-----------|----------|-------|
| LTC2338-18 ADC | 32-bit | Mode 0 | Default SPI2 config |
| DRV8702 x3 | 16-bit | Mode 0 | Switches data width only |
| DAC80508 | 24-bit | Mode 1 | Switches data width + clock phase |
| ADS7066 x3 | 24-bit / 16-bit | Mode 0 | Switches data width only |

`DAC80508_SPITransfer()` sequence:
1. Disable SPI2
2. Set data width to 24-bit, clock phase to CPHA=1 (Mode 1)
3. Re-enable SPI2
4. Assert nCS, transfer, deassert nCS
5. Disable SPI2
6. Restore 32-bit data width, CPHA=0 (Mode 0)
7. Re-enable SPI2

**Concurrency constraint:** Do not interleave LTC2338, DRV8702, DAC80508, or ADS7066 SPI calls. Currently safe (no RTOS, all sequential).

## Register Map

| Register | Address | R/W | Description |
|----------|---------|-----|-------------|
| NOOP | 0x00 | — | No operation |
| DEVICE_ID | 0x01 | R | Device identification |
| SYNC | 0x02 | R/W | Synchronous update control |
| CONFIG | 0x03 | R/W | Power-down and configuration |
| GAIN | 0x04 | R/W | Reference divider and buffer gain |
| TRIGGER | 0x05 | W | Software LDAC trigger / soft reset |
| BRDCAST | 0x06 | W | Broadcast data to all channels |
| STATUS | 0x07 | R | Device status |
| DAC0–DAC7 | 0x08–0x0F | R/W | Channel 0–7 data registers |

## CONFIG Register (0x03)

| Bit | Name | Default | Description |
|-----|------|---------|-------------|
| 8 | REF_PWDWN | 0 | 1 = Power down internal reference |
| 7:0 | DACx_PWDWN | 0x00 | 1 = Power down individual channels |

## GAIN Register (0x04)

| Bit | Name | Default | Description |
|-----|------|---------|-------------|
| 8 | REFDIV-EN | 0 | 1 = Enable internal reference divider (VREF/2) |
| 7:0 | BUF-GAINx | 0x00 | 0 = Gain of 1, 1 = Gain of 2 (per channel) |

## TRIGGER Register (0x05)

| Value | Action |
|-------|--------|
| Bit 4 set | Software LDAC trigger (update sync'd channels) |
| 0x000A | Software reset (all registers to defaults) |

## API Reference

### `DAC80508_Init(DAC80508_Handle *handle)`

Initialise nCS pin and mark handle as ready. SPI2 must already be initialised. Error code `0x40` on failure.

### `DAC80508_WriteReg(handle, addr, data)`

Write 16-bit data to a register. Temporarily reconfigures SPI2 to 24-bit Mode 1.

### `DAC80508_ReadReg(handle, addr, &data_out)`

Read 16-bit data from a register. Sends two SPI transactions (read command + NOOP to clock out response).

### `DAC80508_SetChannel(handle, channel, value)`

Write a 16-bit value to a single DAC channel (0–7). `0x0000` = 0V, `0xFFFF` = full scale.

### `DAC80508_SetAll(handle, value)`

Write the same 16-bit value to all 8 channels via the broadcast register.

### `DAC80508_SoftReset(handle)`

Soft-reset the DAC. All registers return to power-on defaults.

### `DAC80508_ReadDeviceID(handle, &id_out)`

Read the DEVICE_ID register for identification/verification.

## Return Codes

```c
typedef enum {
    DAC80508_OK           = 0,   /* Success */
    DAC80508_ERR_NOT_INIT = 1,   /* Handle not initialised */
    DAC80508_ERR_TIMEOUT  = 2,   /* SPI transfer timeout */
    DAC80508_ERR_SPI      = 3,   /* SPI bus error */
    DAC80508_ERR_PARAM    = 4,   /* Invalid parameter */
} DAC80508_Status;
```

## Output Voltage Calculation

```
V_OUT = (DAC_CODE / 65536) x V_REF x GAIN
```

Where:
- `DAC_CODE` = 16-bit value written to DAC channel register (0–65535)
- `V_REF` = Reference voltage (internal 2.5V or external)
- `GAIN` = 1x (BUF-GAINx = 0) or 2x (BUF-GAINx = 1)

If REFDIV-EN = 1, the internal reference is divided by 2 before the buffer, so effective VREF = 1.25V. With GAIN = 2x, the output swing is restored to 0–2.5V.
